## Recommended Pricing Tiers

### Tier 1: Free (Freemium)

| Feature | Included |
|---------|----------|
| Refinance calculator | Unlimited |
| Basic alerts | 1 alert (email only) |
| Educational content | Yes |

**Goal:** Acquire users, build email list
**Expected:** 80% of users

### Tier 2: Pro ($19/month)

| Feature | Included |
|---------|----------|
| Alerts | Unlimited |
| Notifications | SMS + email |
| Rate updates | Priority (instant vs daily) |
| Calculator features | Advanced |
| Rate trend charts | Yes |

**Goal:** Core revenue stream
**Expected:** 15% of users

### Tier 3: Premium ($49/month)

| Feature | Included |
|---------|----------|
| Everything in Pro | Yes |
| Expert consultation | 1 call/month |
| Personalized recommendations | Yes |
| Early access | New features |
| Support level | White-glove |

**Goal:** High-value customers
**Expected:** 5% of users

### Enterprise: White-Label (Custom Pricing)

| Feature | Included |
|---------|----------|
| Target | Mortgage brokers/banks |
| Branding | Branded calculator + alerts |
| API access | Yes |
| Pricing | $500-2000/month per broker |

**Goal:** B2B revenue stream

## Pricing Experiments to Run

### Test 1: Price Point Testing

| Variant | Price | Measure |
|---------|-------|---------|
| Control | $19/mo | Baseline |
| Variant A | $14.99/mo | Conversion rate x revenue |
| Variant B | $24.99/mo | Conversion rate x revenue |
| Variant C | $29/mo | Conversion rate x revenue |

### Test 2: Annual vs Monthly

| Option | Price | Effective Monthly |
|--------|-------|-------------------|
| Annual | $199/year | $16.58/mo (2 months free) |
| Monthly | $19/mo | $19/mo |

**Measure:** Cash flow, retention

### Test 3: Add-On Revenue

| Add-On | Price | Measure |
|--------|-------|---------|
| Base subscription | $19/mo | Baseline |
| SMS add-on | +$5/mo | Attach rate |
| Expert consult | +$30/session | Total ARPU |

Show Premium ($49) first to make Pro ($19) feel like a deal.

**Pricing page order:**

## Key Metrics to Track

1. **Conversion Rate** - Free to paid conversion
2. **ARPU** - Average revenue per user
3. **LTV** - Customer lifetime value
4. **CAC** - Customer acquisition cost
5. **Churn Rate** - Monthly subscription cancellations
6. **Attach Rate** - Add-on adoption percentage
