# Content Production Playbook

## Overview

This playbook establishes a sustainable system for ongoing blog content production at RefiAlert. Following this workflow, the team produces approximately 10 posts per month (publishing every 3 days).

## Publishing Schedule

| Day | Activity |
|-----|----------|
| Monday | Publish Post #1 |
| Thursday | Publish Post #2 |
| Sunday | Publish Post #3 |

**Optimal publish time:** 8:00 AM ET (highest engagement window)

## Content Pipeline Phases

### Phase 1: Planning (Days 1-7)

**Research & Topic Selection**
- Review keyword research backlog
- Analyze competitor content gaps
- Check trending topics in mortgage/finance space
- Verify topic aligns with editorial calendar

**Keyword Research**
- Primary keyword (search volume > 500/month)
- Secondary keywords (3-5 related terms)
- Long-tail variations for FAQ sections
- Tools: Ahrefs, SEMrush, or free alternatives (Ubersuggest)

**Outline Creation**
- H1 headline (60 characters max)
- H2/H3 structure
- Key points per section
- Data/statistics to include
- Internal link opportunities

**Image Sourcing**
- Featured image (1200x630px for social sharing)
- In-content images (800px width)
- Infographics where applicable
- Sources: Unsplash, Pexels, Canva (for custom graphics)

### Phase 2: Writing (Days 8-14)

**Content Brief Handoff**
- Assign to writer (AI or human)
- Provide outline and research
- Set word count target (1500-2500 words)
- Include style guide reference

**Draft Creation**
- Follow outline structure
- Include all required elements (see Writer Guidelines)
- Cite sources properly
- Add placeholder text for CTAs

**Internal Review**
- Fact-check all statistics
- Verify calculator integrations work
- Check brand voice consistency
- Review for technical accuracy

**Editing & Revisions**
- Grammar and spelling check (Grammarly)
- Readability optimization (Hemingway Editor, target 60+)
- SEO optimization (keyword density, meta tags)
- Final approval

### Phase 3: Production (Days 15-21)

**CMS Setup**
- Create post in CMS
- Set publish date/time
- Configure categories and tags
- Add featured image

**Image Optimization**
- Compress images (<100KB each)
- Add alt text with keywords
- Set proper dimensions

**SEO Finalization**
- Meta title (60 chars)
- Meta description (155 chars)
- URL slug optimization
- Schema markup if applicable

**Distribution Queue**
- Schedule social media posts
- Queue newsletter inclusion
- Set up any paid promotion

## Production Models

### Option A: AI-Generated Content

**Process:**
1. Human creates detailed outline
2. AI (GPT-4/Claude) generates draft
3. Human editor reviews and fact-checks
4. Final polish and publish

**Cost:** ~$50/month (API costs)

**Best for:** High-volume, data-driven content

**Quality controls:**
- Always fact-check AI-generated statistics
- Verify all external links
- Run plagiarism check (Copyscape)
- Add human perspective/anecdotes

### Option B: Freelance Writers

**Process:**
1. Provide content brief
2. Writer submits first draft
3. Editor reviews
4. Writer revisions
5. Final approval

**Cost:** $500-1000/month (10 posts x $50-100/post)

**Best for:** Expert opinion pieces, complex topics

**Sourcing:**
- Contently
- ClearVoice
- Upwork (vetted writers)
- Industry-specific job boards

### Option C: Hybrid Approach (Recommended)

**Process:**
1. AI generates research summary and outline
2. Human writer expands with expertise
3. AI assists with SEO optimization
4. Human editor final review

**Cost:** $200-300/month

**Best for:** Balancing quality and efficiency

## Performance Tracking

### Monthly Content Report

Track these metrics for each post:

| Metric | Target | Tool |
|--------|--------|------|
| Organic traffic | 100+ visits/month after 90 days | GA4 |
| Avg. time on page | > 3 minutes | GA4 |
| Bounce rate | < 65% | GA4 |
| Keyword rankings | Top 20 within 60 days | Search Console |
| Backlinks acquired | 1-2 per post | Ahrefs |
| Social shares | 10+ | Social platforms |
| Calculator conversions | Track signups | Custom events |

### Quarterly Analysis

- Identify top 5 performing posts
- Analyze common themes/topics
- Review underperforming content
- Adjust editorial calendar based on data

## Content Refresh Strategy

### 6-Month Audit Checklist

For each post older than 6 months:

- [ ] Statistics still accurate?
- [ ] Links still working?
- [ ] Information still current?
- [ ] New keywords to target?
- [ ] Competitor content improved?
- [ ] New internal links available?

### Refresh Priorities

1. **High traffic, outdated info** - Update immediately
2. **Ranking positions 5-15** - Optimize to reach top 3
3. **Declining traffic** - Analyze and refresh
4. **Seasonal content** - Update before peak season

### Update Process

1. Export current version (archive)
2. Research new developments
3. Update statistics and data
4. Add new sections if needed
5. Refresh meta description
6. Update publish date (if major changes)
7. Re-promote on social media

## Emergency Procedures

### Content Not Ready for Publish Date
1. Check evergreen content queue for backup
2. Republish/refresh older popular post
3. Adjust social/email schedule
4. Document delay reason for process improvement

### Factual Error Discovered Post-Publish
1. Correct immediately (don't unpublish)
2. Add editor's note if major
3. Notify email subscribers if they received
4. Update social posts if possible
