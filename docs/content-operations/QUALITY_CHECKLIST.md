## Pre-Writing Checklist

- [ ] Content brief approved
- [ ] Primary keyword researched (volume, difficulty)
- [ ] Secondary keywords identified (3-5)
- [ ] Competitor content reviewed
- [ ] Data sources identified
- [ ] Outline approved

---

## Content Quality

### Accuracy
- [ ] All statistics are from reliable sources
- [ ] Statistics are dated within last 6 months
- [ ] All claims are fact-checked
- [ ] Sources are properly cited
- [ ] Rate information is current
- [ ] No outdated year references

### Completeness
- [ ] All sections from brief are complete
- [ ] Word count meets requirement (1500-2500)
- [ ] All user questions answered
- [ ] Clear conclusion with actionable takeaway

### Originality
- [ ] Passed Copyscape plagiarism check
- [ ] Unique angle/perspective provided
- [ ] Original examples or analysis included
- [ ] Properly attributed quotes

---

## Readability

### Metrics
- [ ] Flesch Reading Ease: 60+ (Hemingway Editor)
- [ ] Average sentence length: <20 words
- [ ] No paragraph longer than 4 sentences
- [ ] Grade level: 8th-9th or lower

### Structure
- [ ] Short, punchy introduction hooks reader
- [ ] Clear logical flow throughout
- [ ] Transitions between sections
- [ ] Key points are scannable
- [ ] Conclusion summarizes main points

### Formatting
- [ ] Single H1 (post title only)
- [ ] H2s for major sections
- [ ] H3s for subsections
- [ ] Lists for 3+ related items
- [ ] Tables for comparisons/data
- [ ] Call-out boxes for key info

---

## SEO Optimization

### On-Page SEO
- [ ] Primary keyword in title (H1)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in meta description
- [ ] Primary keyword in URL slug
- [ ] Secondary keywords in H2/H3 headings
- [ ] Keywords used naturally (1-2% density)
- [ ] No keyword stuffing

### Meta Data
- [ ] Title: 60 characters or less
- [ ] Meta description: 155 characters or less
- [ ] Meta description includes CTA
- [ ] URL slug is clean and keyword-rich

### Links
- [ ] Minimum 3 internal links to other posts
- [ ] 1 link to calculator (where relevant)
- [ ] 2-3 external links to authoritative sources
- [ ] Anchor text is descriptive (not "click here")
- [ ] External links open in new tab
- [ ] All links tested and working

---

## Images & Media

### Featured Image
- [ ] Relevant to content
- [ ] Dimensions: 1200x630px minimum
- [ ] File size: <100KB
- [ ] Alt text includes keyword naturally
- [ ] Proper licensing (Unsplash, Pexels, or created)

### In-Post Images
- [ ] All images support content understanding
- [ ] All images have descriptive alt text
- [ ] All images optimized (<100KB each)
- [ ] Images are mobile-responsive
- [ ] Charts/graphs are readable at mobile size

---

## Technical Quality

### Grammar & Spelling
- [ ] Grammarly check completed
- [ ] No spelling errors
- [ ] Consistent punctuation style
- [ ] No run-on sentences

### Style Consistency
- [ ] Brand voice guidelines followed
- [ ] Consistent terminology used
- [ ] Numbers formatted consistently
- [ ] Dates formatted consistently (Month DD, YYYY)

### Accessibility
- [ ] Proper heading hierarchy
- [ ] Alt text on all images
- [ ] Sufficient color contrast
- [ ] Tables have header rows
- [ ] No content hidden from screen readers

---

## Calls-to-Action

- [ ] Clear primary CTA at end of post
- [ ] CTA is relevant to content topic
- [ ] Calculator linked where appropriate
- [ ] Email signup opportunity included
- [ ] Related posts suggested

---

## Final Checks

### Mobile Preview
- [ ] Content displays correctly on mobile
- [ ] Images don't break layout
- [ ] Tables are scrollable/responsive
- [ ] CTAs are tappable
- [ ] No horizontal scrolling required

### Cross-Browser
- [ ] Chrome rendering correct
- [ ] Safari rendering correct
- [ ] Firefox rendering correct

### Legal/Compliance
- [ ] No personal financial advice claims
- [ ] Appropriate disclaimers included
- [ ] No copyrighted material misused
- [ ] GDPR compliant if applicable

## Post-Publish Checklist

### Day of Publish
- [ ] Post is live and accessible
- [ ] All links working on live site
- [ ] Images loading properly
- [ ] Social media posts scheduled
- [ ] Submitted to Google Search Console

### Week 1
- [ ] Monitor for errors/issues
- [ ] Check initial traffic metrics
- [ ] Respond to any comments
- [ ] Track social engagement

### Month 1
- [ ] Review traffic and ranking
- [ ] Check for featured snippet opportunities
- [ ] Add to weekly newsletter if performing
- [ ] Update internal links from older posts
