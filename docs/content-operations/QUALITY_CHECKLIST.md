# Content Quality Checklist

Use this checklist for every piece of content before publishing. All items must be checked for a post to go live.

---

## Pre-Writing Checklist

Complete before starting draft:

- [ ] Content brief reviewed and understood
- [ ] Primary keyword identified (volume: ___, difficulty: ___)
- [ ] Secondary keywords listed (3-5 terms)
- [ ] Competitor content analyzed (2-3 posts)
- [ ] Outline approved
- [ ] Target word count set: _____ words
- [ ] Deadline confirmed: _____

---

## Content Quality Checks

### Structure

- [ ] H1 headline present (only one)
- [ ] H1 is 60 characters or less
- [ ] H1 includes primary keyword
- [ ] Introduction is 150-200 words
- [ ] Introduction includes hook in first sentence
- [ ] H2 headings every 200-300 words
- [ ] Heading hierarchy logical (H2 → H3 → H4)
- [ ] Conclusion includes summary
- [ ] Conclusion includes clear CTA
- [ ] Word count meets target (1500-2500 words)

### Readability

- [ ] Hemingway Editor score: 60+ (actual: ___)
- [ ] Paragraphs are 3-4 sentences max
- [ ] No walls of text
- [ ] Bulleted/numbered lists used appropriately
- [ ] Transition words between sections
- [ ] Read aloud - flows naturally

### Accuracy

- [ ] All statistics have sources
- [ ] Sources are less than 2 years old
- [ ] Facts verified with authoritative sources
- [ ] No unsubstantiated claims
- [ ] Legal/regulatory info verified
- [ ] Calculator examples produce correct results

### Grammar & Style

- [ ] Grammarly check complete (0 errors)
- [ ] Spelling verified (especially proper nouns)
- [ ] Consistent tense throughout
- [ ] Active voice predominant
- [ ] No jargon without explanation
- [ ] Brand voice guidelines followed

### Originality

- [ ] Copyscape check passed
- [ ] No copied sentences/paragraphs
- [ ] Original examples/scenarios used
- [ ] Unique angle/perspective included

---

## SEO Optimization Checks

### Keywords

- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword density 1-2%
- [ ] Secondary keywords naturally included
- [ ] No keyword stuffing

### Meta Tags

- [ ] Meta title written: _____
- [ ] Meta title is 60 chars or less
- [ ] Meta title includes primary keyword
- [ ] Meta description written: _____
- [ ] Meta description is 155 chars or less
- [ ] Meta description includes primary keyword
- [ ] Meta description includes CTA

### URL

- [ ] URL slug is short (3-5 words)
- [ ] URL includes primary keyword
- [ ] URL uses hyphens (not underscores)
- [ ] URL is lowercase

### Links

- [ ] 3-5 internal links included
- [ ] Internal links use descriptive anchor text
- [ ] Internal links open in same tab
- [ ] 2-3 external authority links included
- [ ] External links open in new tab
- [ ] All links tested and working
- [ ] No broken links

---

## Image & Media Requirements

### Featured Image

- [ ] Featured image selected
- [ ] Dimensions: 1200x630px (or aspect ratio)
- [ ] File size: under 100KB
- [ ] Alt text written (includes keyword if natural)
- [ ] Image is relevant to content
- [ ] Image source/license verified

### In-Content Images

- [ ] Images support the content
- [ ] Each image has alt text
- [ ] Images compressed (<100KB each)
- [ ] Images are 800px width max
- [ ] Infographics used where helpful
- [ ] Screenshots are current/accurate

### Image Sources
- [ ] Stock photos from Unsplash/Pexels (free license)
- [ ] Custom graphics from Canva
- [ ] Screenshots captured fresh
- [ ] No copyrighted images without license

## Post-Publish Checklist

Complete within 24 hours of publishing:

### Verification

- [ ] Post displays correctly on desktop
- [ ] Post displays correctly on mobile
- [ ] All images load properly
- [ ] All links work
- [ ] Calculator/tools function correctly
- [ ] No formatting issues
- [ ] Comments enabled (if applicable)

### Distribution

- [ ] Social media posts scheduled
  - [ ] Twitter
  - [ ] LinkedIn
  - [ ] Facebook (if used)
- [ ] Newsletter queue updated (if weekly digest)
- [ ] Internal links added from relevant older posts
- [ ] Search console URL inspection submitted

### Analytics

- [ ] Google Analytics tracking confirmed
- [ ] Custom events working (calculator usage, etc.)
- [ ] Goal tracking set up
- [ ] Add to content performance tracker

## Quick Reference: Common Fixes

### Low Hemingway Score
- Shorten sentences
- Remove adverbs
- Use simpler words
- Break up paragraphs

### Missing Internal Links
- Search site for related content
- Link from intro to tools/calculators
- Reference related guides in body

### Weak CTA
- Be specific ("Use our calculator" not "Learn more")
- Create urgency without being pushy
- Match CTA to content topic
