# RefiAlert Writer Guidelines

Standards and requirements for all blog content contributors.

---

## Brand Voice

### Tone
- **Authoritative but approachable** - We're experts, but not condescending
- **Empathetic** - Understand that refinancing decisions are stressful
- **Action-oriented** - Help readers make informed decisions
- **Data-driven** - Back claims with statistics and sources

### Voice Characteristics

| Do | Don't |
|----|-------|
| "You could save $200/month by refinancing now" | "Homeowners may potentially realize savings" |
| "Let's break down the numbers" | "The following analysis demonstrates" |
| "Here's what matters" | "It is important to note that" |
| "Rate dropped 0.25% today" | "A rate reduction of 25 basis points occurred" |

### Writing Style
- **Second person** ("you", "your") for engagement
- **Active voice** preferred over passive
- **Short paragraphs** (2-4 sentences max)
- **Conversational** but professional
- **Avoid jargon** or explain it when necessary

## Content Structure

### Standard Post Structure

Every post should include:

1. **Headline** (60 characters max)
   - Include primary keyword
   - Promise clear value
   - Use numbers when appropriate ("7 Ways to...")

2. **Meta Description** (155 characters max)
   - Summarize post value
   - Include primary keyword
   - End with call-to-action when possible

3. **Featured Image** (1200x630px)
   - Relevant to content
   - Optimized size (<100KB)
   - Include alt text with keyword

4. **Introduction** (150-200 words)
   - Hook the reader immediately
   - State the problem/question
   - Promise what they'll learn
   - Include primary keyword in first 100 words

5. **Body** (1500-2500 words)
   - Use H2, H3 headings for structure
   - Include data and statistics
   - Add real-world examples
   - Link to our calculator/tools
   - Break up text with lists, tables, images

6. **Conclusion** (100-150 words)
   - Summarize key points
   - Clear call-to-action (CTA)
   - Link to relevant tools/calculator

### Heading Hierarchy

```
H1: Post Title (one per post)
  H2: Main Sections
    H3: Subsections
      H4: If needed (rare)
```

- One H1 per page (the title)
- Use H2 for major sections
- Use H3 for subsections within H2
- Include keywords in headings naturally

## SEO Requirements

### Keyword Usage

| Element | Keyword Placement |
|---------|------------------|
| Title (H1) | Include primary keyword |
| First 100 words | Primary keyword |
| URL slug | Primary keyword (hyphenated) |
| Meta description | Primary keyword |
| H2 headings | 1-2 secondary keywords |
| Image alt text | Primary or secondary keyword |
| Throughout body | Natural usage, no stuffing |

**Keyword Density**: Aim for 1-2% for primary keyword (natural usage)

### Internal Links

Every post must include:
- **Minimum 3 internal links** to other blog posts
- **1 link to calculator** (where relevant)
- Use descriptive anchor text (not "click here")

Example:
- Good: "Use our [refinance calculator](/calculator) to see your break-even point"
- Bad: "Click [here](/calculator) to calculate"

### External Links

Every post should include:
- **2-3 links to authoritative sources**
- Acceptable sources: Fed, CFPB, major lenders, academic research
- Open external links in new tab
- Use rel="noopener" for security

## Quality Standards

### Accuracy

- **Fact-check all statistics** before submission
- **Cite sources** for all data claims
- **Verify rate information** against current data
- **Use recent data** (within 6 months when possible)

Trusted Data Sources:
- Federal Reserve (FRED)
- Freddie Mac Primary Mortgage Market Survey
- Mortgage Bankers Association
- Consumer Financial Protection Bureau
- Bureau of Labor Statistics

### Readability

Target **Flesch Reading Ease score of 60+** (8th-9th grade level)

How to achieve:
- Short sentences (15-20 words average)
- Short paragraphs (2-4 sentences)
- Use simple words over complex ones
- Break up dense information with lists
- Explain technical terms when first used

**Test your content**: Paste into Hemingway Editor or use Grammarly

### Originality

- **Zero plagiarism** - All content must be original
- Run through Copyscape before submission
- Properly attribute quotes and data
- Paraphrase research findings in your own words

## Formatting Guidelines

### Lists

Use bullet points for:
- Non-sequential items
- Features or benefits
- Quick reference items

Use numbered lists for:
1. Step-by-step instructions
2. Ranked items
3. Sequential processes

### Tables

Use tables for:
- Comparing options
- Data presentation
- Quick reference information

Example:
| Loan Type | Best For | Typical Terms |
|-----------|----------|---------------|
| Fixed Rate | Stability | 15, 20, 30 years |
| ARM | Short-term savings | 5/1, 7/1, 10/1 |

### Call-out Boxes

Use for:
- Important warnings
- Pro tips
- Key takeaways

Format:
```markdown
> **Pro Tip**: [Important advice here]

> **Warning**: [Caution or important note]

> **Key Takeaway**: [Main point to remember]
```

### Images

Requirements:
- **Dimensions**: 1200px width minimum
- **File size**: Under 100KB (use TinyPNG)
- **Format**: WebP preferred, PNG for graphics, JPG for photos
- **Alt text**: Descriptive, include keyword when natural

Types of images to include:
- Featured image (required)
- Charts/graphs for data
- Screenshots of calculator
- Infographics for complex concepts

## Content Types

### Rate Analysis Posts

Focus: Analyze current rate trends and implications

Structure:
1. Current rate summary
2. Recent trend analysis
3. Factors driving changes
4. What it means for borrowers
5. Action recommendations

Example: "Fed Holds Rates Steady: What It Means for Mortgage Borrowers"

### Regional Guides

Focus: Country/region-specific refinancing information

Structure:
1. Market overview
2. Regulatory environment
3. Major lenders and options
4. Step-by-step process
5. Common pitfalls
6. Resources and contacts

Example: "Refinancing in Canada: A Complete 2026 Guide"

### How-To Guides

Focus: Practical, actionable instructions

Structure:
1. What you'll learn/accomplish
2. Prerequisites/requirements
3. Step-by-step instructions
4. Common mistakes to avoid
5. Next steps

Example: "How to Calculate Your Refinance Break-Even Point"

### Case Studies

Focus: Real examples of refinancing decisions

Structure:
1. Situation overview
2. Homeowner profile
3. Options considered
4. Decision factors
5. Results and lessons
6. Key takeaways

Note: Use composite/anonymized examples, not real customer data

## Submission Checklist

Before submitting your draft:

### Content
- [ ] Word count meets requirement (1500-2500)
- [ ] All sections from brief are complete
- [ ] Statistics are current and sourced
- [ ] Examples are relevant and clear

### SEO
- [ ] Primary keyword in title, first 100 words, meta description
- [ ] H2/H3 headings include secondary keywords
- [ ] Minimum 3 internal links included
- [ ] 2-3 external authoritative links included
- [ ] Meta description is 155 characters or less

### Quality
- [ ] Hemingway score 60+ (readability)
- [ ] Grammarly check completed
- [ ] No plagiarism (Copyscape clear)
- [ ] All facts verified

### Formatting
- [ ] Proper heading hierarchy (H1 > H2 > H3)
- [ ] Short paragraphs (2-4 sentences)
- [ ] Lists used for 3+ items
- [ ] Images have alt text

## Revision Process

### Round 1: Content Review
Editor checks for:
- Accuracy of information
- Completeness against brief
- Logical flow and structure
- Voice and tone alignment

### Round 2: SEO Review
SEO lead checks for:
- Keyword optimization
- Internal/external links
- Meta data optimization
- Technical SEO elements

### Round 3: Final Polish
Final review for:
- Grammar and spelling
- Formatting consistency
- Image optimization
- CTA effectiveness
