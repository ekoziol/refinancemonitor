# Financial Logic Review

Review mortgage and financial calculations for correctness:

## Calculation Accuracy
- Monthly payment formulas (principal + interest)
- Amortization schedule calculations
- Interest rate conversions (APR vs monthly)
- Refinance savings calculations
- Break-even analysis logic

## Business Logic
- Alert trigger conditions and thresholds
- Refinance cost accounting
- Rate comparison logic
- Payment estimation accuracy

## Edge Cases
- Zero or negative values handling
- Very large loan amounts
- Unusual term lengths
- Rate boundary conditions
- Rounding and precision issues

## Data Consistency
- Currency formatting consistency
- Percentage vs decimal handling
- Date calculations for loan terms

Verify calculations match industry-standard formulas.
