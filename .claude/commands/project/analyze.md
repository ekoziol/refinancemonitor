# Project Analysis

Analyze the current project structure and provide insights:

1. **Codebase Health Check**:
   - Review recent commits and identify any technical debt
   - Check for outdated dependencies or security vulnerabilities
   - Analyze code complexity and maintainability

2. **Architecture Assessment**:
   - Evaluate current domain structure and separation of concerns
   - Identify potential architectural improvements
   - Check for adherence to established patterns

3. **Performance Analysis**:
   - Review database queries and identify potential bottlenecks
   - Analyze frontend bundle size and loading performance
   - Check for any memory leaks or inefficient algorithms

4. **Documentation Review**:
   - Verify CLAUDE.md is up-to-date with recent changes
   - Check if new features are properly documented
   - Identify missing documentation that should be added

5. **Testing Coverage**:
   - Analyze test coverage across all applications
   - Identify critical paths that need additional testing
   - Suggest improvements to testing strategy

Focus on actionable insights and prioritize by impact and effort required.
